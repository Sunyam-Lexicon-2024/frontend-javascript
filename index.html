Bish Bosh 2.0

A small html content generator built to study basic Javascript operations. This Bish-Bosh generator takes a Bish number, a Bosh Number and a Bish Bosh Row count.
the generator caluclates the modulo(%) of each given number and if it resolves to 0 outputs the corresponding text reply in the display field.
The Row count denotes how many iterations are produced, starting from 1 (i.e. from 1 to the given row count number).
